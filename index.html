YouTube-videoer hentes via offentlige Piped-instanser (360p/720p). Direkte videolenker fungerer hvis serveren tillater det (CORS).

Hvis YouTube-henting feiler

Siden kjører uten egen server (GitHub Pages er kun statiske filer), så YouTube-videoer hentes via offentlige proxy-tjenester som av og til er nede eller blokkert. Reserveløsning: last ned videoen med yt-dlp på din egen maskin og last opp filen øverst.
